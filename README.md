### Assign 6.6 Polyline and Polygon
This is an app that displays a park and hiking trail using Google Maps and the polyline and polygon. When clicking on either the park or the trail, you are provided with additional information and you are able to customize the size and color of both the polyline and polygon. 

In order to run the app, you will need to modify the "app/src/main/res/values/google_maps_api.xml" file with you api key.
